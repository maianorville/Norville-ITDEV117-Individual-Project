# Norville-ITDEV117-Individual-Project

What it is
Study Session Planner is a simple app that helps students organize their study time. It lets you create sessions, add tasks, and track how long you study so you can stay focused and productive.

The problem it solves
A lot of students start studying without a clear plan and end up losing track of time or skipping tasks. This app fixes that by giving you a timer and checklist so you can see exactly what you’ve done and how long you’ve worked.

Who it’s for
It’s made for students who want to manage their study sessions better — especially those balancing multiple classes or assignments.

Main features
Create and name your own study sessions
Add tasks and set a timer for each session
Mark tasks complete as you go
View a summary of total time and completed tasks
Simple, easy‑to‑read interface

How it works
You start by creating a new study session and entering your tasks. Once you hit “Start Session,” the timer begins. As you finish each task, you check it off. When the timer ends, the app shows a summary of what you accomplished and how long you studied.
